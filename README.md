# Genetics, Evolution and Bioinformatics utilities package

### Modules:

- cross
  - classes to create pedigrees/organisms and peform crosses
- popdist
  - classes to analyse Population Distributions (allele frequencies etc)
- utils
  - string_utils
     - BWT, SuffixArray etc
  - file_utils
     - read_fasta, etc
  - gen_utils   
     - general utilities
